# 💻 Meu Portfólio Pessoal - Caitano Vitor

Este repositório contém o código-fonte do meu portfólio profissional, desenvolvido para consolidar meus projetos de desenvolvimento de software e minha trajetória na área de tecnologia.

## 🚀 Tecnologias e Ferramentas
O projeto utiliza tecnologias fundamentais de desenvolvimento web para garantir performance e uma interface intuitiva:

* **HTML5**: Estruturação semântica de conteúdo.
* **CSS3**: Estilização moderna e layout responsivo (Mobile First).
* **JavaScript**: Adição de interatividade e lógica de navegação.

## 🛠️ Principais Funcionalidades
- **Design Responsivo**: Adaptado para visualização em smartphones, tablets e desktops.
- **Seção de Projetos**: Vitrine interativa com links para outros repositórios.
- **Sobre Mim**: Resumo da minha formação em Análise e Desenvolvimento de Sistemas (ADS).
- **Contato Direto**: Integração com ícones de redes sociais e formulário.

## 📂 Como Executar o Projeto Localmente
Para visualizar o site na sua máquina, siga estes passos:

1.  **Clone o repositório:**
    ```bash
    git clone (https://github.com/CaitanoVitor/PORTIFOLIO.git)
    ```
2.  **Acesse a pasta:**
    ```bash
    cd PORTIFOLIO
    ```
3.  **Abra o arquivo:**
    Execute o arquivo `index.html` em qualquer navegador atual.
